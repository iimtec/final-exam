# Ordering Application (final_project)

#Project overview
This project is using the OOP for the usage of online ordering. Which will show the driver and the customer name, order summary, and order status. 

---

## Features
- Customer and driver introduction -> showing their names.
- Order summary -> show the order information including item, customer's name,  status and driver's name.
- Delivery status -> show that the driver is now delivering the ordered items and the vehicle that they use.
- Final status -> show that the delivery is successful.

---

## How to run
Open the terminal in the project folder and run:
```
python creature_simulation.py
```
